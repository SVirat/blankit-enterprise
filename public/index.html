Call the API

Send a POST request with your text or upload a document. You get back clean output plus a full redaction map.

Terminal
$ curl -X POST https://api.Blankit.dev/v1/redact \
  -H "Authorization: Bearer Blankit_live_sk_7f3a..." \
  -H "Content-Type: application/json" \
  -d '{"text": "John Smith, SSN 123-45-6789"}'

The API returns redacted text with safe placeholders, plus a map so you can restore originals client-side. The LLM never sees real data.
